📄 Personal Webpage - Documentation

📘 About the Project

This is a simple personal webpage showing your name, job title, skills, profile picture, email, and social media links.

📁 Files

  • index.html — main page structure and text.
  • styles.css — styling (colors, fonts, layout).
  • images/ — folder with your profile picture and social icons.

🛠️ How to Edit

1. Change Text

Open index.html and edit inside the section with class maininformation:

  • Name: Edit the text inside the <h3> tag.
  • Job title: Edit the first <p> under the name.
  • Skills: Edit the paragraph with class p2.
  • Description: Edit the paragraph with class p3.

2. Change Profile Picture

In index.html, find the <img> with class profile:

<img src="images/Zivver.jpeg" alt="" class="profile" />

Replace Zivver.jpeg with your picture file in the images/ folder.

3. Change Email

Find the <a> tag with class gmail:

<a href="https://mail.google.com/mail/?view=cm&to=wvwxxii@gmail.com" class="gmail">wvwxxii@gmail.com</a>

Edit both the email link and visible email text.

4. Change Social Links and Icons

Update the links (href) to your social media:

<a href="https://github.com" target="_blank">
  <img src="images/icons8-github-logo-24.png" alt="GitHub" />
</a>

Replace icon images or update the paths in images/.

➕ Adding More Info or Text

To add content, open index.html and place new sections inside <section class="Container">.

Then adjust dimensions in styles.css:

.main {
  width: 290px;
  height: 420px;
}

.Container {
  width: 250px;
  height: 390px;
}

💡 How to Use

  • Keep all files in one folder.
  • Open index.html in your browser.
  • Make edits and refresh to view changes.

🔁 Reusing the Card Only

  1. Copy the HTML inside the <section class="main"> block.
  2. Copy related CSS from styles.css:
    • .main, .Container, .profile
    • .maininformation, .gmail, .contact
  3. Replace or copy required images.

📌 Notes

  • Use consistent file names.
  • Use optimized image sizes.
